## 📗 DESCRIPTION

Anemos is a client-side mod that gives you true fullbright by overriding Minecraft’s brightness cap (raised from 100 to 1500). It removes all darkness and shadows, making everything fully lit — regardless of time, biome, or dimension.

Vanilla gamma doesn’t work with shaders? No worries. It includes a fake night vision system that works flawlessly even with shaders enabled.

Got hit with Blindness or Darkness? The bypass system cancels those visual effects so you can see normally — though it's not recommended on servers without proper permissions!

## ✨ KEY FEATURES

* Maximum visibility — also known as fullbright.
* A fake night vision toggle that supports shaders.
* Bypass for Blindness and Darkness.
* Smooth transition effects.
* Deep customization — without breaking things.

## 🧩 USAGE

* Press `G` to toggle between max gamma and previous gamma.
* Press `N` to simulate night vision.
* Press `B`/`M` to bypass Blindness/Darkness effects.

Open the config screen via Mod Menu or run the `/configure anemos` command to adjust features.
